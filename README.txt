1. Копируем скачанные файлы в домашнюю папку. Обязательно в домашнюю папку, иначе работать не будет.

2. Проверяем, что файл GPUcontrol является исполняемым: 
	ПКМ — Свойства — Права — Должна стоять галочка после слова «Выполнение» 
	(Разрешить выполнение файла как программы). 

3. Чтобы команды через SUDO в скрипте выполнялись без запроса пароля:
	запускаем терминал и в нем пишем 
	
	sudo visudo
	
	нажимаем Enter и у вас попросит пароль от учетной записи, вводим его и ниже строк
	
	# Allow members of group sudo to execute any command
	%sudo   ALL=(ALL:ALL) ALL
	
	вводим такую строку
	
	m1 ALL=(ALL) NOPASSWD: ALL
		где  m1 имя вашего пользователя
	
	чтобы сохранить изменения нажимаем  CTRL+O, Enter
	для выхода из редактора  CTRL+X

4. Добавляем скрипт в автоагрузку. 
	В убунту заходим в «настройки запуска приложений» 
	нажимаем кнопку добавить 
	Пишем имя, которое вам удобно (я пишу GPUcontrol)
	нажимаем кнопку обзор и указываем путь к файлу 
	описание можно не писать, и нажимаем добавить.

5. Открываем файл GPUcontrol в текстовом редакторе и меняем значения как вам необходимо:
	sleep 30	задержка запуска скрипта в секундах
	templimit=65	ограничение максимальной температуры видеокарт в градусах Цельсия	
   Сохраняем (CTRL+S)

6. Пользуемся с удовольствием!
